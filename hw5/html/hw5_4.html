Problem 4

(a) Given an array AAA of nnn integers, find an efficient algorithm to compute the largest sum of a continuous run.

(b) Now assume that there are nnn numbers (some possibly negative) on a circle, and we wish to find the maximum contiguous sum along an arc of the circle. Give an efficient algorithm for solving this problem.

(b)

We extend the solution to (a) to the circular case by simply concatenating a copy of the array with itself, or

A[i]={A[i]i<nA[i−n]i≥n,0≤i<2nA[i] = \begin{cases} A[i] & i < n \\ A[i-n] & i \geq n \end{cases}, \quad 0 \leq i < 2n A[i]={A[i]A[i−n]​i<ni≥n​,0≤i<2n
