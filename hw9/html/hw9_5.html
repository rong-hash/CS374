Problem 5

Are the following problems decidable or undecidable? If the language is decidable, explain why, and if it's not, prove it

(a) L5a={⟨M⟩∣ M is a TM that accepts the string 1011 in ∣1011∣6 steps}L_{5a} = \left\{ \langle M \rangle \mid \text{ M is a TM that accepts the string } 1011 \text{ in } |1011|^6 \text{ steps} \right\}L5a​={⟨M⟩∣ M is a TM that accepts the string 1011 in ∣1011∣6 steps}

(b) L5b={⟨M⟩∣ M is a TM that does not accept any strings in ∣w∣6 steps}L_{5b} = \left\{ \langle M \rangle \mid \text{ M is a TM that does not accept any strings in } |w|^6 \text{ steps} \right\}L5b​={⟨M⟩∣ M is a TM that does not accept any strings in ∣w∣6 steps}

(a)

L5aL_{5a}L5a​ is decidable. We can simulate the TM MMM with a counter. If the counter reaches ∣1011∣6|1011|^6∣1011∣6 before the TM halts, we reject the TM, otherwise we accept it.

(b)

L5bL_{5b}L5b​ is undecidable.

Suppose there is an algorithm DecideL5b\text{Decide}L_{5b}DecideL5b​ that correctly decides the language L5bL_{5b}L5b​. Then we can solve the halting problem as follows:

  • Then M′M'M′ accepts an input string xxx in ∣x∣6|x|^6∣x∣6 steps (during the simulation of ⟨M,w⟩\langle M, w \rangle⟨M,w⟩).
  • So DecideL5b\text{Decide}L_{5b}DecideL5b​ rejects the encoding ⟨M′⟩\langle M' \rangle⟨M′⟩.

  • So DecideL5b\text{Decide}L_{5b}DecideL5b​ accepts the encoding ⟨M′⟩\langle M' \rangle⟨M′⟩.
