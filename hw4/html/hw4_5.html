Problem 5

Given an arbitrary array A[1:n]A[1:n]A[1:n], describe an algorithm to determine in O(n)O(n)O(n) time whether AAA contains more than n4\frac{n}{4}4n​ copies of any value. Do not use hashing, or radix sort, or any other method that depends on the precise input values.


Solution

Intuition

  • In a sorted array, elements with the same value occupy a contiguous subarray. Such a subarray with length over n4\frac{n}{4}4n​ must cover at least one of the array's three quarters.
  • In other words, if there is a value that appears more than n4\frac{n}{4}4n​ times, it must be among the three quarters. This also applies to unsorted arrays.
