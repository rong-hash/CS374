Problem 3

Let BBB and CCC be languages over Σ={0,1}\Sigma = \{0, 1\}Σ={0,1}. Define:

B→0C={w∈C∣∃x∈B,#(0,w)=#(0,x)}B \overset{0}{\rightarrow} C = \{w \in C \mid \exists x \in B, \#(0, w) = \#(0, x)\} B→0C={w∈C∣∃x∈B,#(0,w)=#(0,x)}

Show that the class of regular languages is closed under the →0\overset{0}{\rightarrow}→0 operation.


Solution

Proof. We show that if BBB and CCC are regular, then B→0CB \overset{0}{\rightarrow} CB→0C is regular. Suppose that BBB and CCC can be represented by two NFAs, namely MB=(QB,Σ,δB,sB,AB)M_B = (Q_B, \Sigma, \delta_B, s_B, A_B)MB​=(QB​,Σ,δB​,sB​,AB​) and MC=(QC,Σ,δC,sC,AC)M_C = (Q_C, \Sigma, \delta_C, s_C, A_C)MC​=(QC​,Σ,δC​,sC​,AC​).

We first construct a "swapped" NFA to our convenience:

MBS=(QB,Σ,δBS,sB,AB)M_{BS} = (Q_B, \Sigma, \delta_{BS}, s_B, A_B) MBS​=(QB​,Σ,δBS​,sB​,AB​)

, where ∀qB∈QB\forall q_B \in Q_B∀qB​∈QB​,

  • δBS(qB,0)={δB(qB,0)}\delta_{BS}(q_B, 0) = \{\delta_B(q_B, 0)\}δBS​(qB​,0)={δB​(qB​,0)}
  • δBS(qB,1)={qB}\delta_{BS}(q_B, 1) = \{q_B\}δBS​(qB​,1)={qB​}
  • δBS(qB,ε)={δB(qB,1),δB(qB,ε)}\delta_{BS}(q_B, \varepsilon) = \{\delta_B(q_B, 1), \delta_B(q_B, \varepsilon)\}δBS​(qB​,ε)={δB​(qB​,1),δB​(qB​,ε)}

Since every state has an implicit ε\varepsilonε-transition to itself, we are effectively changing the inputs of the self-loops to 111. Correspondingly, all the arcs labelled 111 are now labelled ε\varepsilonε.

Such swapping turns BBB into a purely "counting" machine, where it pauses when seeing a 111 and uses the ε\varepsilonε-transitions to "make up" for the moves. In this case, any string w∈Σ∗w \in \Sigma^*w∈Σ∗ whose number of 000's equals to that of a string x∈Bx \in Bx∈B, even if www is not accepted by MBM_BMB​, will be accepted by MBSM_{BS}MBS​.

We then construct an NFA M′M'M′ that accepts B→0CB \overset{0}{\rightarrow} CB→0C:

M′=(Q′,Σ,δ′,s′,A′)M' = (Q', \Sigma, \delta', s', A') M′=(Q′,Σ,δ′,s′,A′)

, where

  • Q′=QB×QCQ' = Q_B \times Q_CQ′=QB​×QC​
  • δ′((qB,qC),a)=δBS(qB,a)×δC(qC,a),∀qB∈QB,qC∈QC,a∈Σ\delta'((q_B, q_C), a) = \delta_{BS}(q_B, a) \times \delta_C(q_C, a), \forall q_B \in Q_B, q_C \in Q_C, a \in \Sigmaδ′((qB​,qC​),a)=δBS​(qB​,a)×δC​(qC​,a),∀qB​∈QB​,qC​∈QC​,a∈Σ
  • s′=(sB,sC)s' = (s_B, s_C)s′=(sB​,sC​)
  • A′={q1×q2∣q1∈AB,q2∈AC}A' = \{q_1 \times q_2 \mid q_1 \in A_B, q_2 \in A_C\}A′={q1​×q2​∣q1​∈AB​,q2​∈AC​}

M′M'M′ is also known as the cross-product machine of MBSM_{BS}MBS​ and MCM_CMC​.

A string ending up in an accepting state (q1,q2)(q_1, q_2)(q1​,q2​) has two implications: (1) q1∈ABq_1 \in A_Bq1​∈AB​ so that the string has an acceptable number of 000's and (2) q2∈ACq_2 \in A_Cq2​∈AC​ so that the string is in the language CCC. Q.E.D.
