Problem 4

Suppose we are given an array A[1:n]A[1:n]A[1:n] of nnn integers, which could be positive, negative, or zero, sorted in increasing order so that A[1]≤A[2]≤⋯≤A[n]A[1] \le A[2] \le \cdots \le A[n]A[1]≤A[2]≤⋯≤A[n].

Suppose we wanted to count the number of times some integer value xxx occurs in AAA. Describe an algorithm (as fast as possible) which returns the number of elements containing value xxx.


Solution

Intuition. We use binary search to locate the leftmost and rightmost occurrences of xxx in AAA. The number of occurrences is the difference between bounds.
