Problem 3

Describe the context-free grammar that describes each of the following languages:

(a) All strings in {0,1}∗\{0, 1\}^*{0,1}∗ whose length is divisible by 5.

(b) L3b={0i1j2i+j∣i,j≥0}L_{3b} = \{ 0^i 1^j 2^{i+j} \mid i, j \geq 0 \}L3b​={0i1j2i+j∣i,j≥0}

(c) L3c={0i1j2k∣i=j or j=k}L_{3c} = \{ 0^i 1^j 2^k \mid i = j \textrm{ or } j = k \}L3c​={0i1j2k∣i=j or j=k}

(d) L3d={w∈{0,1}∗∣#(01,w)=#(10,w)}L_{3d} = \{ w \in \{0, 1\}^* \mid \#(01, w) = \#(10, w) \}L3d​={w∈{0,1}∗∣#(01,w)=#(10,w)}


Solution

In each case, the context-free grammar is defined as

G=(V,T,P,S)G = (V, T, P, S) G=(V,T,P,S)

(a) Recursively append groups of 5 bits.

  • V={S,A}V = \{ S, A \}V={S,A}
  • T={0,1}T = \{ 0, 1 \}T={0,1}
  • P={S→ϵ∣ASA→(0+1)5P = \begin{cases} S \rightarrow \epsilon \mid AS \\ A \rightarrow (0+1)^5 \end{cases}P={S→ϵ∣ASA→(0+1)5​

(b) Symmetrically grow the "outer shell" before growing the "inner shell."

  • V={S,A}V = \{ S, A \}V={S,A}
  • T={0,1,2}T = \{ 0, 1, 2 \}T={0,1,2}
  • P={S→A∣0S2A→ϵ∣1A2P = \begin{cases} S \rightarrow A \mid 0S2 \\ A \rightarrow \epsilon \mid 1A2 \end{cases}P={S→A∣0S2A→ϵ∣1A2​

(c) There are 2 cases.

  • V={S,A,B,C,D}V = \{ S, A, B, C, D \}V={S,A,B,C,D}
  • T={0,1,2}T = \{ 0, 1, 2 \}T={0,1,2}
  • P={S→A∣BA→A2∣CB→0B∣DC→ϵ∣0C1D→ϵ∣1D2P = \begin{cases} S \rightarrow A \mid B \\ A \rightarrow A2 \mid C \\ B \rightarrow 0B \mid D \\ C \rightarrow \epsilon \mid 0C1 \\ D \rightarrow \epsilon \mid 1D2 \end{cases}P=⎩⎨⎧​S→A∣BA→A2∣CB→0B∣DC→ϵ∣0C1D→ϵ∣1D2​

(d) # rising edge = # falling edge. Edges are created in pairs.

  • V={S,A,B,C,D}V = \{ S, A, B, C, D \}V={S,A,B,C,D}
  • T={0,1}T = \{ 0, 1 \}T={0,1}
  • P={S→ϵ∣A∣BA→C∣CBCB→D∣DADC→0∣0CD→1∣1DP = \begin{cases} S \rightarrow \epsilon \mid A \mid B \\ A \rightarrow C \mid CBC \\ B \rightarrow D \mid DAD \\ C \rightarrow 0 \mid 0C \\ D \rightarrow 1 \mid 1D \end{cases}P=⎩⎨⎧​S→ϵ∣A∣BA→C∣CBCB→D∣DADC→0∣0CD→1∣1D​
