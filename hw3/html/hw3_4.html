Problem 4

An all-NFA MMM is a 5-tuple (Q,Σ,δ,q0,F)(Q, \Sigma, \delta, q_0, F)(Q,Σ,δ,q0​,F) that accepts x∈Σ∗x \in \Sigma^*x∈Σ∗ if every possible state that MMM could be in after reading input xxx is a state from FFF, in contrast to an ordinary NFA that accepts a string if some state among these possible states is an accept state.

Solution

Proof. Given a typical NFA N=(Q,Σ,δ,s,A)N = (Q, \Sigma, \delta, s, A)N=(Q,Σ,δ,s,A), we can transform it into a DFA

D=(Q′,Σ,δ′,s′,A′)D = (Q', \Sigma, \delta', s', A') D=(Q′,Σ,δ′,s′,A′)

where

The four definitions above also applies to the transformation of an all-NFA, except for A′A'A′ which must be refined as

A′={X⊆Q∣X⊆A and X≠∅}A' = \{X \subseteq Q | X \subseteq A \textrm{ and } X \neq \emptyset\} A′={X⊆Q∣X⊆A and X=∅}

since (1) XXX must be non-empty and (2) XXX must be a subset of AAA. This corresponds to the definition of an all-NFA that every possible ending state should be an accept state.
