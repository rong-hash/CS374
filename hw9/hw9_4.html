Problem 4

Let T={⟨M⟩∣ M is a TM that accepts wR whenever it accepts w}T = \left\{ \langle M \rangle \mid \text{ M is a TM that accepts } w^R \text{ whenever it accepts } w \right\}T={⟨M⟩∣ M is a TM that accepts wR whenever it accepts w}. Show that TTT is undecidable.


Solution

Suppose there is an algorithm DecideT\text{DecideT}DecideT that correctly decides the language TTT. Then we can solve the halting problem as follows:

DecideHalt(⟨M,w⟩)Encode the following Turing machine M′:M′(x)if  x=01return Truerun M on input wreturn Trueif  DecideT(⟨M′⟩)return Trueelsereturn False\text{DecideHalt}(\langle M, w \rangle) \\ \qquad \text{Encode the following Turing machine } M': \\ \qquad \qquad M'(x) \\ \qquad \qquad \qquad \text{if } \ x = 01 \\ \qquad \qquad \qquad \qquad \textbf{return } \text{True} \\ \qquad \qquad \qquad \text{run } M \text{ on input } w \\ \qquad \qquad \qquad \textbf{return } \text{True} \\ \qquad \textbf{if } \ \text{DecideT}(\langle M' \rangle) \\ \qquad \qquad \textbf{return } \text{True} \\ \qquad \textbf{else} \\ \qquad \qquad \textbf{return } \text{False} \\ DecideHalt(⟨M,w⟩)Encode the following Turing machine M′:M′(x)if  x=01return Truerun M on input wreturn Trueif  DecideT(⟨M′⟩)return Trueelsereturn False

We prove this reduction correct as follows.

If. Suppose MMM halts on input www.

  • Then M′M'M′ accepts every input string xxx.
  • In particular, M′M'M′ accepts wRw^RwR whenever it accepts www.
  • So DecideT(⟨M′⟩)\text{DecideT}(\langle M' \rangle)DecideT(⟨M′⟩) accepts the encoding ⟨M′⟩\langle M' \rangle⟨M′⟩.
  • So DecideHalt\text{DecideHalt}DecideHalt correctly accepts the encoding ⟨M,w⟩\langle M, w \rangle⟨M,w⟩.

Only if. Suppose MMM does not halt on input www.

  • Then M′M'M′ diverges on every input string xxx except 010101.
  • In particular, M′M'M′ does not accept wRw^RwR whenever it accepts www -- a counterexample is w=01w = 01w=01 and wR=10w^R = 10wR=10.
  • So DecideT(⟨M′⟩)\text{DecideT}(\langle M' \rangle)DecideT(⟨M′⟩) rejects the encoding ⟨M′⟩\langle M' \rangle⟨M′⟩.
  • So DecideHalt\text{DecideHalt}DecideHalt correctly rejects the encoding ⟨M,w⟩\langle M, w \rangle⟨M,w⟩.

Therefore, DecideHalt\text{DecideHalt}DecideHalt is correct. But this contradicts the fact that the halting problem is undecidable. Thus, DecideT\text{DecideT}DecideT is incorrect, and TTT is undecidable. ■\blacksquare■
