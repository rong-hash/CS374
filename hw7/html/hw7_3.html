Problem 3

Your job is to arrange nnn schildren in a straight line, facing front. You are given a list of mmm statements of the form "iii hates jjj," which indicates that iii cannot be put somewhere behind jjj.

(a) Give an algorithm that orders the line (or says that it is not possible) in O(m+n)O(m + n)O(m+n) time.

(b) Suppose instead you want to arrange the children in rows such that if iii hates jjj, then iii must be in a lower numbered row than jjj. Give an efficient algorithm to find the minimum number of rows needed, if possible.


Solution

(a)

First we construct a graph based on the relation of hatred, where

  • each node represents a student, and
  • an edge u→vu \to vu→v represents that the student uuu hates another student vvv.

Then we do a topological sort on the graph. The node rank gives a order of the students.

  • Note that the algorithm fails if a cycle is detected in the topological sorting procedure. A legal arrangement is not possible in this case.

(b)

Starting from the "source" with topological rank 0, we perform a BFS on the graph and find the maximum depth (or the longest path from the source). The algorithm reports failure when backward edges (i.e., cycles) are detected.

Alternatively, we can use Kahn's algorithm to find the topological order. The number of rows is the length of the longest path.

Pseudocode.

  • Construct 2 queues, current\text{current}current and next\text{next}next
  • Add all vertices with no incoming edges to current\text{current}current
  • Initialize an empty list to store the rows
  • While current\text{current}current and next\text{next}next is not empty
    • construct a new row\text{row}row
    • while current\text{current}current is not empty
      • Dequeue a vertex www from current\text{current}current and add it to row\text{row}row
      • For each vertex xxx adjacent to www
        • Remove the edge x→wx \to wx→w
        • If www has no incoming edges, add it to next\text{next}next
      • Swap current\text{current}current and next\text{next}next
      • Add row\text{row}row to the list of rows

If all edges have been removed from the graph, the topological order is successfully found, and the rows are constructed. Having remaining edges indicates a cycle, in which case a topological order is not possible.
