Problem 1

A strongly independent set is a subset of vertices SSS in a graph GGG such that for any two vertices in SSS, there is no path of length two in GGG. Prove that Strongly Independent Set is NP-hard.


Solution

We reduce the strongly independent set problem to the independent set problem.

Reduction. For an arbitrary undirected graph G=(V,E)G = \left(V, E\right)G=(V,E), we construct a new undirected graph G′G'G′ by

  • adding a new vertex on every edge of GGG, and
  • connecting two "edge" vertices if the edges share a common vertex.

Formally,

G′=(V′,E′), whereV′=V∪{vij∣(i,j)∈E}E′={(i,vij),(j,vij)∣(i,j)∈E}∪{(vik,vjk)∣(i,k),(j,k)∈E} \begin{aligned} G' &= \left(V', E'\right) \text{, where} \\ V' &= V \cup \left\{v_{ij} \mid \left(i, j\right) \in E\right\} \\ E' &= \left\{ \left(i, v_{ij}\right), \left(j, v_{ij}\right) \mid \left(i, j\right) \in E\right\} \cup \left\{\left(v_{ik}, v_{jk}\right) \mid \left(i, k\right), \left(j, k\right) \in E \right\} \end{aligned} G′V′E′​=(V′,E′), where=V∪{vij​∣(i,j)∈E}={(i,vij​),(j,vij​)∣(i,j)∈E}∪{(vik​,vjk​)∣(i,k),(j,k)∈E}​

We claim that GGG has an independent set if and only if G′G'G′ has a strongly independent set.

If. Suppose GGG has an independent set SSS.

  • Then the path any two vertices i,j∈Si, j \in Si,j∈S has length at least 2 in GGG, say i→k→ji \to k \to ji→k→j.
  • Therefore, the path between iii and jjj in G′G'G′ has length at least 3 -- that is, i→vik→vjk→ji \to v_{ik} \to v_{jk} \to ji→vik​→vjk​→j.

This confirms that G′G'G′ has a strongly independent set.

Only if. Suppose G′G'G′ has a strongly independent set S′S'S′. We can swap every "edge" vertex in S′S'S′ with a "vertex" vertex such that S′S'S′ remains a strongly independent set in G′G'G′.

  • Suppose the minimum distance to vijv_{ij}vij​ from ∀u∈S′−vij\forall u \in S' - v_{ij}∀u∈S′−vij​ is ddd.

    • If the path does not go through vijv_{ij}vij​, then the path length is at least ddd (change vik→vijv_{ik} \to v_{ij}vik​→vij​ to vik→iv_{ik} \to ivik​→i)
