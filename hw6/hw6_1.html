Problem 1

Largest Square of 111's. You are given a n×nn \times nn×n bitonic array AAA and the goal is to find the set of elements within that array that form a square filled with only 111’s.


Solution

Memoization. We define a function LS(i,j),i,j∈[1,n+1]LS(i, j), i, j \in [1, n+1]LS(i,j),i,j∈[1,n+1]. It records the size of the largest square of 111's that can be formed with the top-left corner at (i,j)(i, j)(i,j). The matrix is filled from bottom-right to top-left.
