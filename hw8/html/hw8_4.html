Problem 4

SAT reductions.

(a) Stingy SAT. Given a set of clauses (each a disjunction of literals) and an integer kkk, find a satisfying assignment in which at most kkk variables are True\text{True}True, if such an assignment exists. Prove that Stingy SAT is NP-hard.

(b) Double SAT. The problem asks whether a given satisfiability problem has at least two different satisfying assignments. Prove that Double SAT is NP-hard.


Solution

(a)

To prove that Stingy SAT is NP-hard, we need to (1) prove that Stingy SAT is an NP problem, and (2) reduce the SAT problem to a Stingy SAT problem.

Pre-condition. Let (F,k)(F, k)(F,k) be an instance of Stingy SAT, where FFF is an instance of SAT with kkk variables. We need to show that a set of assignments XXX can make (F,k)(F, k)(F,k) a real solution in polynomial time, which proves that Stingy SAT is an NP problem.

Target. We need to create a certificate that converts SAT to Stingy SAT, i.e., XXX is the solution of FFF if and only if XXX is the solution of (F,k)(F, k)(F,k). SAT⇔Stingy SAT\text{SAT} \Leftrightarrow \text{Stingy SAT}SAT⇔Stingy SAT.

Proof of adequacy. If XXX is the solution of FFF, then at most kkk variables are True\text{True}True in XXX. Therefore, XXX satisfies (F,k)(F, k)(F,k), which means XXX is the solution of (F,k)(F, k)(F,k).

Proof of necessity. Assuming XXX is the solution of (F,k)(F, k)(F,k), it implies that XXX assigns at most kkk variables to True\text{True}True, and also satisfies FFF. Hence, XXX is the solution of FFF. ■\blacksquare■
