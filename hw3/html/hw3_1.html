Problem 1

A finite-state transducer (FST) gives an output based on the transition instead of the current state. It is defined by a 5-tuple:

(Σ,Γ,Q,δ,s)(\Sigma, \Gamma, Q, \delta, s) (Σ,Γ,Q,δ,s)

The output alphabet of a FSTARFST_{AR}FSTAR​ consists of two signals, namely accept and reject (Γ={A,R}\Gamma = \{\textrm{A}, \textrm{R}\}Γ={A,R}). We say that L(FSTAR)L(FST_{AR})L(FSTAR​) represents the language consisting of all strings that end with an accept (A\textrm{A}A) output signal.

Prove that L(FSTAR)L(FST_{AR})L(FSTAR​) represents the class of regular languages.


Solution

Proof. For the given FST, we construct a DFA M′M'M′ such that L(M′)=L(FSTAR)L(M') = L(FST_{AR})L(M′)=L(FSTAR​):

M′=(Σ,Q′,δ′,s′,A′)M' = (\Sigma, Q', \delta', s', A') M′=(Σ,Q′,δ′,s′,A′)

where

  • Q′=Q×ΓQ' = Q \times \GammaQ′=Q×Γ
  • δ′((q,b),a)=δ(q,a),∀q∈Q,a∈Σ,b∈Γ\delta'((q, b), a) = \delta(q, a), \forall q \in Q, a \in \Sigma, b \in \Gammaδ′((q,b),a)=δ(q,a),∀q∈Q,a∈Σ,b∈Γ
  • s′=(s,A)s' = (s, \textrm{A})s′=(s,A)
  • A′={(q,A)∣q∈Q}A' = \{ (q, \textrm{A}) \mid q \in Q \}A′={(q,A)∣q∈Q}

Here we assume that in a FST, δ:Q×Σ→Q×Γ\delta: Q \times \Sigma \rightarrow Q \times \Gammaδ:Q×Σ→Q×Γ

The core idea is to explicitly encode the last output signal in the states. Note that there is a clear boundary between next-state logic and output logic -- thus, the expression of δ′\delta'δ′ is unrelated to bbb.
