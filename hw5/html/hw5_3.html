Problem 3

Given an n×mn \times mn×m grid filled with non-negative numbers, find a path from top left (1,1)(1,1)(1,1) to bottom right (n,m)(n,m)(n,m) that minimizes the sum of all numbers along its path. You can only move either down (++i)(++i)(++i) or right (++j)(++j)(++j) at any point in time. What is the running time of your algorithm?


Solution

Recurrence function. We denote the grid as GGG and the DP matrix as MMM. The DP recurrence is

Memoization. M(i,j)M(i, j)M(i,j) represents the minimum sum of all numbers along the optimal path from (1,1)(1, 1)(1,1) to (i,j)(i, j)(i,j). The matrix should be filled top-to-bottom, left-to-right. A grid can only be filled after its left and upper neighbors are filled.

The optimal path is obtained by backtracking from the bottom-right corner (n,m)(n, m)(n,m) to the top-left corner (1,1)(1, 1)(1,1), choosing the neighbor with the minimum sum at each step. Time complexity is O(nm)O(nm)O(nm).
